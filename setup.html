If you already have Claude Code installed, this is the fastest way to get started with BizBrain OS. Don't have Claude Code yet? Start with the full install guide instead.

Created by William Welsh at Tech Integration Labs — refined over years of real-world iteration, experimentation, and daily use across dozens of businesses and projects. BizBrain OS is the missing context layer that makes AI genuinely useful for business.

How It Works

1

Open Your Terminal

Navigate to where you want your Brain folder created. cd ~/Documents works great.

2

Start Claude Code

Type claude and hit enter. You're in.

3

Paste the Prompt

Click the copy button above, paste into Claude Code, and hit enter.

4

Answer & Watch

The wizard interviews you, builds your Brain, analyzes your docs, and suggests custom workflows.

What You'll Get in Five Minutes

This isn't a toy demo. It's the same system powering real businesses every day.

◆ Structured Business Memory

Client folders, project tracking, operations, knowledge base — all organized so AI knows where everything is, every session.

◆ AI Brain (CLAUDE.md)

A personalized context file that teaches AI your business, clients, preferences, and workflows. It reads this automatically, every time.

◆ 7 Slash Commands

/inbox, /todo, /client, /status, /find, /organize, /report — your business toolkit, ready on day one.

◆ Smart Document Analysis

Point it at your existing files and watch it understand your business, extract insights, and suggest how to organize everything.

◆ Git Backup

Version-controlled and backed up to GitHub. Every change tracked, recoverable, and accessible from anywhere.

◆ Custom AI Suggestions

After learning about your business, the wizard suggests specific workflows, automations, and capabilities tailored to you.
