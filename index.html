Capabilities

What It Does

BizBrain OS is a complete business operating system that lives in your terminal. Here's what makes it different.

Voice-First Setup

Talk about yourself, your business, your clients. BizBrain learns everything through a natural voice interview and configures itself around your world.

Modular Configuration

27+ modules covering everything from CRM to content creation. Pick what you need, skip what you don't. Each module installs independently.

Local-First & Private

Your business data never leaves your machine. No cloud databases, no third-party storage. Everything runs locally with full encryption support.

Chrome Automation

Connect GitHub, Netlify, social profiles, and more. BizBrain automates browser interactions to set up and manage your online presence.

Smart Dashboard

A visual command center that tracks your setup progress, shows active modules, and provides quick access to every capability in your system.

Auto-Updating

Git pull and your system evolves. New modules, improved agents, better prompts -- all delivered through simple version control. No manual updates.

How It Works

From zero to running your business from the terminal in under 10 minutes.

01

Install with One Command

Run npx create-bizbrain and the installer handles everything -- cloning the repo, checking dependencies, and scaffolding your workspace.

$ npx create-bizbrain

✓ Cloned TechIntegrationLabs/bizbrain-os
✓ Dependencies installed
✓ Workspace initialized
Ready! Run cd bizbrain && claude to begin setup.
02

Voice Interview About Your Business

BizBrain asks about your business, clients, tech stack, and workflows. Speak naturally -- it builds your entire business profile from the conversation.

03

Configure Modules One at a Time

04

Quick Start

Up and Running in 10 Minutes

Everything you need to install, configure, and start using BizBrain OS.

Prerequisites

Claude Code Latest The AI engine that powers everything
Node.js 18+ Runs the local dashboard server
Git Any For cloning and pulling updates
Chrome Latest Optional -- for voice input & credential setup

Installation

Terminal 
# Option 1: One-command install
$ npx create-bizbrain

# Option 2: Git clone (for contributors)
$ git clone https://github.com/TechIntegrationLabs/bizbrain-os.git
$ cd bizbrain-os

# Then open Claude Code and run:
$ /setup

# Dashboard launches at http://localhost:5555

After Setup

Daily Usage 
$ /bb1 entity add-client "Acme Corp"  # Add a client
$ /hours                               # Track time
$ /gsd new my-project                  # Plan a project
$ /content blog-post "Weekly wins"     # Generate content
$ /status                              # Check everything
$ /todo                                # View all tasks
$ /dashboard                           # Reopen dashboard
Architecture

Clean Separation by Design

System files update via git pull. Your data and config are never touched.

Folder Structure 
bizbrain-os/
|
|-- .bizbrain/              # SYSTEM (git-tracked, auto-updated)
|   |-- modules/            #   27 module definitions (JSON configs)
|   |-- dashboard/          #   Local web UI (Node.js + vanilla JS)
|   |-- wizard/             #   Setup interview & generators
|   |-- agents/             #   AI agent definitions
|   |-- commands/           #   Slash command definitions
|   +-- templates/          #   Entity & project templates
|
|-- config.json             # YOUR CONFIG (gitignored)
|-- CLAUDE.md               # YOUR CONTEXT (AI-generated)
|-- Clients/                # YOUR DATA (gitignored)
|-- Partners/
|-- Vendors/
|-- Projects/
|-- Knowledge/
+-- Brand/                  # Logo, colors, assets
Update Without Fear 
$ git pull origin main
# System files update. Your data and config are untouched.
