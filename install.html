Prerequisites

Before you start, make sure you have these installed. If you're missing any, follow the links to install them — each takes just a minute or two.

Download Node.js →

Any version

Download Git →

Built-in

Open your terminal and run these to confirm everything is installed:

Terminal 
$ node --version

$ git --version

Install Claude Code

Claude Code is the AI engine that powers BizBrain OS. It's a command-line tool made by Anthropic that gives you a powerful AI assistant right in your terminal.

Install via npm

Terminal 
$ npm install -g @anthropic-ai/claude-code
added 1 package in 12s
Permission error on Mac/Linux? If you get an EACCES error, try: sudo npm install -g @anthropic-ai/claude-code

Verify Installation

Terminal 
$ claude --version
claude-code v1.x.x    # you should see a version number
Official docs For the latest installation instructions and troubleshooting, see the Claude Code documentation.
3

Get Your Anthropic API Key

Claude Code needs an API key to work. You'll get this from the Anthropic Console. It takes about 2 minutes.

Create an Account

Go to console.anthropic.com and sign up (or log in if you already have an account).

Add Billing

Navigate to Settings → Billing and add a payment method. Claude Code is pay-per-use — typical usage for BizBrain OS costs a few dollars per month depending on how actively you use it.

Cost Expectations Light usage (a few sessions per day) typically runs $5–15/month. Heavy usage might be $30–50/month. You can set spending limits in the console to control costs.

Create an API Key

  1. In the Anthropic Console, go to Settings → API Keys
  2. Click "Create Key"
  3. Give it a name like bizbrain
  4. Copy the key (starts with sk-ant-...)
Save your key somewhere safe You'll only see the full key once. Store it in a password manager or secure note. Never share it publicly or commit it to Git.

Connect Claude Code to Your Key

The first time you run claude, it will ask for your API key. Paste it in and you're set. Alternatively, you can set it as an environment variable:

~/.zshrc or ~/.bashrc 
# Add this line to your shell config:
export ANTHROPIC_API_KEY="sk-ant-your-key-here"

# Then reload:
$ source ~/.zshrc
PowerShell 
# Set for current session:
> $env:ANTHROPIC_API_KEY = "sk-ant-your-key-here"

# Or set permanently (System Environment Variables):
# Settings > System > About > Advanced > Environment Variables
# Add ANTHROPIC_API_KEY with your key value
Test it Run claude in your terminal. If you see a prompt waiting for input, you're connected and ready to go.

What Happens During Installation

No matter which method you choose, the setup process will:

  • ✓ Ask you about your business, clients, projects, and goals
  • ✓ Create your complete folder structure (Clients, Projects, Operations, Knowledge, etc.)
  • ✓ Generate a personalized CLAUDE.md — the AI brain config that Claude reads every session
  • ✓ Set up slash commands (/inbox, /todo, /client, /status, and more)
  • ✓ Optionally analyze your existing documents and learn about your business
  • ✓ Suggest custom AI workflows tailored to your specific situation
  • ✓ Set up Git backup (optional)

Your First Session

Once setup is complete, here's how to start using your BizBrain every day:

Daily usage 
# Navigate to your Brain folder
$ cd ~/BizBrain   # or wherever you created it

# Start Claude Code
$ claude

# Claude automatically reads your CLAUDE.md and knows everything.
# Try these commands:

> /status          # see your business dashboard
> /todo            # view your to-do list
> /todo add Follow up with client about proposal
> /client Acme     # look up a client
> /inbox           # process files you dropped in _inbox/
> /find invoice    # search across all your files

Things You Can Just Say

Beyond slash commands, you can talk naturally:

  • ✓ "Add a note to Acme Corp: Met with Sarah, she wants the proposal by Friday"
  • ✓ "Help me write an email to the team about the project deadline"
  • ✓ "What are my priorities for today?"
  • ✓ "Research competitors in the [your industry] space"
  • ✓ "Create a new project for the website redesign"
  • ✓ "Commit my changes" (saves a snapshot to Git)
The more you use it, the better it gets Every file you add, every note you save, every client interaction you record — it all compounds. Your AI assistant gets more useful every single day because it has more context to draw from.

Next Steps

You're up and running. Here's a suggested timeline for getting the most out of BizBrain OS:

Week 1–2: Build the Habit

  • ✓ Start every work session by opening Claude in your Brain folder
  • ✓ Use /status each morning to see where you left off
  • ✓ Use /todo add throughout the day to capture tasks
  • ✓ Drop documents in _inbox/ and run /inbox to organize them

Week 3–4: Expand

  • ✓ Set up Git & GitHub for automatic backups
  • ✓ Add your first MCP plugin (Brave Search is the easiest)
  • ✓ Update your CLAUDE.md as your business evolves

Month 2+: Go Deeper

  • ✓ Create custom slash commands for your specific workflows
  • ✓ Explore the full BizBrain OS modules (27+ available)
  • ✓ Connect to OpenClaw for autonomous AI agents
  • ✓ Join the community on GitHub Discussions
