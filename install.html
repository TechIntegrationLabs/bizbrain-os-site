Standalone Installer

Download a single file for your OS. No Node.js, no Git, no prerequisites. The installer handles everything:

  • ✓ Auto-installs Git, Node.js, and Claude Code if missing
  • ✓ Walks you through a quick business onboarding (name, industry, tools)
  • ✓ Downloads and configures BizBrain OS with your info pre-filled
  • ✓ Optionally import existing business files via drag-and-drop
  • ✓ Generates a personalized setup prompt for Claude Code
🖥 Download for Windows

How to run it

  1. Click the download button above
  2. Open your Downloads folder
  3. Double-click bizbrain-setup.exe
  4. If Windows SmartScreen appears, click "More info" then "Run anyway"
  5. Follow the on-screen prompts — the installer does the rest

Easiest: paste one line into Terminal

Open Terminal (search "Terminal" in Spotlight or find it in Applications → Utilities) and paste this:

Terminal
curl -fsSL https://bizbrain-os.com/install.sh | bash

This automatically downloads the right version for your Mac, handles permissions, and starts the installer. Nothing is installed without your knowledge.

What does this command do? It downloads a small script from bizbrain-os.com that fetches the installer, makes it runnable, and launches it. You can read the script first if you'd like — it's short and simple.

If you already have Node.js 18+ and Git installed, one command does it all:

Terminal

Install Claude Code

Claude Code is the AI engine that powers BizBrain OS. You need three things: Git, Node.js, and Claude Code itself. Pick your OS below.

1a. Install Git Bash

Git for Windows

Includes Git Bash terminal

Download, run the installer, accept all defaults. Then open Git Bash from your Start menu.

Download Git for Windows →

1b. Install Node.js

Node.js

v18 or newer (LTS recommended)

Download the Windows Installer (.msi), run it, accept defaults.

Download Node.js →

1c. Install Claude Code

Open Git Bash and run:

Git Bash
$ npm install -g @anthropic-ai/claude-code

1d. Log in

Run claude in your terminal. It will walk you through authentication — either with an Anthropic API key or a Claude Pro/Max subscription.

Git Bash
$ claude
Welcome to Claude Code! Let's get you set up...
API Key or Subscription? You can use Claude Code with an Anthropic API key (pay-per-use, ~$5–15/month for light use) or a Claude Pro/Max subscription. Either works.
Node not found? Close and reopen Git Bash after installing Node.js. It needs a fresh terminal to register the PATH.

1a. Install Git

Most Macs already have Git. Check with git --version. If not installed, macOS will offer to install it automatically.

macOS

Built-in or Xcode CLT

Open Terminal, type git --version. macOS installs it if needed.

Linux

Use your package manager

Ubuntu: sudo apt install git
Fedora: sudo dnf install git

1b. Install Node.js

Node.js

v18 or newer (LTS recommended)

Download from nodejs.org, or use Homebrew: brew install node

1c. Install Claude Code

Terminal
$ npm install -g @anthropic-ai/claude-code
Permission error? Try: sudo npm install -g @anthropic-ai/claude-code

1d. Log in

Run claude in your terminal. It will walk you through authentication — either with an Anthropic API key or a Claude Pro/Max subscription.

Terminal
$ claude
Welcome to Claude Code! Let's get you set up...
API Key or Subscription? You can use Claude Code with an Anthropic API key (pay-per-use, ~$5–15/month for light use) or a Claude Pro/Max subscription. Either works.

Verify It Works

$ claude --version
claude-code v1.x.x    # you should see a version number
Claude Code is running? You're ready for Step 2. This is the only prerequisite.

What Happens During Setup

No matter which method you choose, the setup process will:

  • ✓ Ask about your business, clients, projects, and goals
  • ✓ Create your complete folder structure
  • ✓ Generate a personalized CLAUDE.md — the AI brain config that Claude reads every session
  • ✓ Set up slash commands (/inbox, /todo, /client, /status, and more)
  • ✓ Optionally analyze your existing documents and learn about your business
  • ✓ Set up Git backup (optional)
